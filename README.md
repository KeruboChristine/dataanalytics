

PROJECT PROPOSAL

Title:

Water Access & Sanitation Inequality Analysis in Health Care Facilities



1. Introduction

Access to safe water and sanitation in health care facilities is essential for quality healthcare and infection prevention. However, many regions across the world still experience inequalities in these basic services. Using data from the WHO/UNICEF Joint Monitoring Programme (JMP), this project aims to analyze disparities in water and sanitation access across different regions over time.



2. Problem Statement

Despite global efforts to improve WASH services, inequalities persist between regions. Some areas have high levels of basic service access, while others still lack even minimal facilities. This imbalance affects healthcare quality and increases the risk of disease transmission. There is a need to analyze these disparities to identify regions that require urgent intervention.



3. Objectives

General Objective

To analyze inequalities in water and sanitation access in health care facilities across regions.

Specific Objectives

 To examine trends in water and sanitation access over time
 To compare service levels across different regions
 To identify regions with the lowest access to basic services
 To measure inequality gaps between regions
 To provide data-driven recommendations



4. Research Questions

 Which regions have the lowest access to basic water services?
 How has sanitation access changed over time?
 What is the gap between regions with high and low access?
 How does water access compare to sanitation access?
 Which regions show the most improvement over time?



5. Data Description

The study uses secondary data from the WHO/UNICEF Joint Monitoring Programme (JMP).

Dataset Features

 Regional-level data (Global, SDG regions)
 Indicators: Water, Sanitation, Hygiene, Waste Management
 Service levels: Basic, Limited, No Service
 Time series data (multiple years)
 Values expressed as percentages

Data Structure

The dataset is in Excel format with multiple sheets:

 Water
Sanitation
Hygiene
Waste Management
Cleaning

The data is initially in wide format and will be transformed into a tidy (long) format for analysis.

6. Methodology

6.1 Data Cleaning

 Remove unnecessary rows and columns
 Rename variables for consistency
 Convert data into long format
 Handle missing values
 Convert data types (year, percentages)

6.2 Data Analysis Techniques

Descriptive statistics (mean, trends)
Comparative analysis across regions
Inequality measurement (max–min gap)
Trend analysis over time

6.3 Data Visualization

Line graphs (trend analysis)
Bar charts (regional comparison)
Gap analysis charts



7. Expected Outcomes

Identification of regions with poor WASH access
Clear trends showing improvement or decline
Quantified inequality gaps between regions
Insights into differences between water and sanitation access
Evidence-based recommendations



8. Significance of the Study

This study will help:

Policymakers identify priority regions
Improve planning for WASH interventions
Support achievement of (United Nations Sustainable Development Goals), especially SDG 6 (Clean Water and Sanitation)
Enhance healthcare quality through better infrastructure planning



9. Limitations

Data is aggregated (no country-level detail)
Missing values in some years
Does not capture real-time conditions



10. Conclusion

This project will provide a clear understanding of inequalities in water and sanitation access in health care facilities. By analyzing regional trends and gaps, it will highlight areas needing urgent attention and contribute to improving global health outcomes.
