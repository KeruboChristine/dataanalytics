Water Access & Sanitation Inequality Analysis
 Project Overview

This project explores global inequalities in access to clean water and sanitation using data from the WHO/UNICEF Joint Monitoring Programme (JMP). The analysis focuses on identifying disparities across countries and regions, understanding trends over time, and visualizing geographic patterns using maps.

This project emphasizes data cleaning, exploratory data analysis (EDA), and storytelling through visualization.

Objectives

1.Analyze global trends in water and sanitation access over time
2.Compare access levels across regions and countries
3.Identify inequality gaps between regions
4.Visualize geographic disparities using maps
5.Communicate insights clearly using data storytelling

 Dataset

Source: WHO/UNICEF JMP WASH Dataset (Excel file)

Key Features of the Dataset
1.Multi-sheet Excel file
2.Country-level data
3.Yearly observations
4.Multiple indicators for water and sanitation
5.Regional classifications (SDG regions)
